## Chat with channels in real time
***
This project is a chat room in real-time, it uses channels to group users into different places by given topics.

**Stack and technologies used:**
- Frontend
  - Javascript vanilla
- Backend
    - Socket.io
      - namespaces
      - broadcasting
      - event emitters
    - NodeJs
    - Express

Clone the project:

`git clone ...
`    
Install the project:

` npm install`

Run dev environment:

`npm run dev
`
